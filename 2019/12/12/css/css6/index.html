css伪元素、伪类及特性的整理

发表于 2019-12-12 更新于 2019-12-14

伪元素

特性：
虚拟的一个元素，用于向已有的元素添加特殊效果，可用标签元素实现该效果。
一个选择器一般来说只能使用一个伪元素，并且伪元素必须处于选择器语句的最后。

常见的伪元素及特性

  1. ::selection CSS伪元素应用于文档中被用户高亮的部分。

  2. ::first-letter会选中某 block-level element第一行的第一个字母，并且文字所处的行之前没有其他内容。

    • ::first-line CSS pseudo-element 在某block-level element的第一行应用样式。第一行的长度取决于很多因素，包括元素宽度，文档宽度和文本的文字大小。

    • ::first-line 不能匹配任何真实存在的html元素。

    • ::first-line 伪元素只能在块容器中,所以,::first-line伪元素只能在一个display值为block, inline-block, table-cell 或者 table-caption中有用.。在其他的类型中，::first-line 是不起作用的。

  4. 伪元素就是控制需要视觉聚焦的元素后面的背景元素样式的。

伪类

符号 特性
语言
:dir 匹配特定文字书写方向的元素
:lang 基于元素语言来匹配页面元素
表单
:checked 匹配每个选中的输入元素
:default 表示一组相关元素中的默认表单元素
:dsiabled 属性规定禁用按钮
:empty 选择器匹配没有子元素的每个元素
:enabled 匹配每个已启用的元素
:in-range 代表一个<input>元素，其当前值处于属性min 和max 限定的范围之内
:out-of-range 用于标签的值在指定区间之外时显示的样式
:indeterminnate 表示状态不确定的表单元素
:valid 在表单元素的值需要根据指定条件验证时设置指定样式
:invalid 表示任意内容未通过验证的 <input> 或其他 <form> 元素
:optional 表示任意没有required属性的 <input><select> 或 <textarea> 元素使用它
:required 在表单元素是必填项时设置指定样式
:read-only 用于选取设置了 “readonly” 属性的元素
:read-write 用于匹配可读及可写的元素
:scope 表头单元与数据单元相关联的方法
状态
:link 向未访问的链接添加特殊的样式
:visited 表示用户已访问过的链接
:hover 用于选择鼠标指针浮动在上面的元素
:active 让页面能在浏览器监测到激活时给出反馈,当用鼠标交互时，它代表的是用户按下按键和松开按键之间的时间
:focus 用于选取获得焦点的元素
结构化
:not 选取除了指定元素以外的所有元素
:frist-child 用于选取属于其父元素的首个子元素的指定选择器
:frist-of-type 匹配属于其父元素的特定类型的首个子元素的每个元素
:last-child 用来匹配父元素中最后一个子元素
:last-of-type 匹配属于其父元素的特定类型的最后一个子元素的每个元素
:nth-child 匹配父元素中的第 n 个子元素，元素类型没有限制
:nth-last-child 匹配同类型中的倒数第n个同级兄弟元素
:nth-of-type 匹配同类型中的第n个同级兄弟元素
:nth-last-of-type 匹配属于父元素的特定类型的第 N 个子元素的每个元素，从最后一个子元素开始计数
:only-child 匹配属于其父元素的唯一子元素的每个元素
:noly-of-type 匹配属于其父元素的特定类型的唯一子元素的每个元素
:target 代表一个唯一的页面元素，其id与当前URL片段匹配
其他
:root 用匹配文档的根元素
:fullscreen 应用于当前处于全屏显示模式的元素
