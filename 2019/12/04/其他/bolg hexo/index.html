如何使用hexo搭建自己的博客

发表于 2019-12-04 更新于 2019-12-14

准备工作

以Windows 64位为例
1.安装git
首先访问git,其次点击64-bit Git for Windows Setup进行下载，最后下载完成直接进行安装即可。
2.安装node.js
先访问node,
而后下载左边的8.10.0 LTS版本，最后下载完成之后安装即可。
3.注册github账号
打开https://github.com,根据要求进行注册即可。

创建github仓库

新建一个你的用户名.github.io的仓库，创建完成之后。
配置SSH key,首先执行
cd ~/. ssh
检查本机已存在的ssh密钥。
然后执行
ssh-keygen -t rsa -C “邮件地址”
然后连续3次回车，会生成一个文件在用户目录下。
打开用户目录，找到.ssh\id_rsa.pub文件，记事本打开并复制里面的内容。
打开你的github主页，进入个人设置 -> SSH and GPG keys -> New SSH key：将刚复制的内容粘贴到key，保存。
执行
ssh -T git@github.com
检查是否成功，如果提示
Are you sure you want to continue connecting (yes/no)?
输入yes，然后看到：
Hi liuxianan! You've successfully authenticated, but GitHub does not provide shell access.
看到这个信息说明SSH已配置成功！
最后配置 git config --global user.name "liuxianan"
git config --global user.email "xxx@qq.com" 即可。

使用hexo

执行 npm install -g hexo 进行安装。
然后新建一个文件夹，cd 文件夹位置进入文件夹，执行
hexo into
hexo会自动下载一些文件到这个目录，然后继续
hexo g
hexo s
执行以上命令之后，hexo就会在public文件夹生成相关html文件。
hexo s是开启本地预览服务，打开浏览器访问 http://localhost:4000 即可看到内容。
如果你一切都配置好了
hexo d
一定要把ssh key配置好，
在配置_config.yml中有关deploy的部分，我用的Vs Code，写代码的一个软件，直接打开，找到deploy，修改。
deploy:
type: git
repository: 你的仓库地址
branch: master
然后执行
npm install hexo-deployer-git --save
这个的作用是安装一个插件,最后
hexo d
提交所有改动。

备注

以上就是用hexo搭建blog的全部步骤，至于写自己blog文章的软件，可以自己下，也可以直接hexo new “postName”进行编写，因为博主也是一个小白，就能帮你们这么多了，有不对的地方，也希望大家理解。
