JavaScript数据类型及其特性整理

发表于 2019-12-14

基本数据类型

  • number 数字
    数值很显然就是表示数字的值，如整数和小数都是数值。在JavaScript里的数值有很多种表示的方法，例如十进制、十六进制，也可用用科学计数法表示，例如：x=1;x=0.1
  • string字符串
    字符串是就一个或者多个字符排在一起，并且放在单或双引号内的值，如‘abc’“abc”，例如：x=”holle”;x=’word’(注：字符串一旦创建，他们的值是不可改变的，string类型里面还包含着一些特殊的字符字面量，也叫转义序列，例如：\n,\t，\b等等）
  • bloolean布尔类型
    布尔值只有两个值，真（true）和假（false）,在比较运算符、相等运算符和逻辑运算符，就会返回布尔值
  • undefined
    值只有一个，就是自身,可以表示为“没有”，undefined表示一个没有给值的非对象，转为数值时为NaN （NaN是一个非数值且一个特殊的数值，NaN有俩个特点，NaN与任何值都不想等（包括自身），任何涉及NaN的操作都会返回NaN）
  • null
    值只有一个，就是自身，可以表示为“没有”，null表示一个没有给值的对象，转为数值时为 0
