HTML基本知识

发表于 2019-12-05 更新于 2019-12-14

HTML基本结构

HTML：<html>文档</html>
头部：<head>标题、字符集</head>
主体：<boby>文本、图像、视频</boby>

HTML标签

  • <html></html>

    • html标签是由尖括号包括
    • html标签是成对出现的
    • 第一个是开始，第二个结束

  • <br/> 强制换行

  • <hr/> 水平线

  • <center></center> 内容居中显示

    HTML编辑器

  • Notepad

    • 启动软件
    • 编辑文件
    • 保存html
    • 浏览器运行

  • Notepad++

  • Sublime Text3

    HTML基本的标签

  • HTML标题

  • HTML段落

    • <p></p>

  • HTML链接

    • <a>

  • HTML图像

    • <img>

HTML元素与属性

元素: 开始标签到结束标签的所有内容
例: <h1>这是元素!!!</h1>
属性: 属性的形式是以”名称=值”,属性是在HTML开始标签中指定的。
例：<h1 属性="这是属性"></h1>

HTML列表

  • 无序列表
    • <ui> 无序列表标记
    • <li>列表项目标记</li>
    • </ui>
  1. 有序列表
    1. <ol>有序列表标记
    2. <li>列表项</li>
    3. </ol>

HTML图像标记

  1. src：图片的位置与格式

  2. width：图片的宽度

  3. height：图片的高度

  4. border：图片的边框宽度

  5. alt

    • 图片加载完成后，鼠标移动到图片上时，出现文字
    • 使搜索引擎通过文字搜索到此图片

  6. 单标记

HTML超链接

  • <a herf=" " targe=" " name=" "></a>

  • herf：链接地址网页视频图片音乐。

  • name：网页锚点名称

  • target：打开方式

    • _blank：新窗口打开
    • _seif：当前窗口
    • _parent：在父窗口打开
    • _top：在顶层窗口打开

      HTML表格

  • 创建表格

    • 创建表格标签：<table></table>
    • 创建行标签：<tr></tr>
    • 创建列标签：<td></td>

  • 跨行跨列表格

    • colspan=”所跨的列数”
    • rowspan=”所跨的行数”

  • 表格布局

    • cellpadding=”文字与边框的距离”

    • cellspacing=”表格内框的宽度”

    • 数据分组

      • <thead>：对应报表的页眉
      • <thody>：报表的数据主体
      • <tfoot>：报表的页脚部分

    • <caption>：标题

    • <th>：表头

      HTML表单

  • 基本语法

    • 表单的各种元素
    • action=”表单提交地址”
    • method=”提交方式”

  • 元素

    • 文本

      • type=”text”
      • 多行文本框
        • <textarea></textarea>
        • cols=”行数”
        • rows=”列数”

    • 密码框

      • type=”password”

  • 按钮

    • 单选按钮

      • type=”redio”

    • 复选框

      • type=”checkbox”

  • 下拉列表<input>

    • name=”元素名称”

    • type=”类型”

    • value=”值”

    • size=”显示宽度”

    • maxlength=”最大字符长度”

    • checked=”是否选中”

    • 列表框

      • <select>显示选项</select>
      • <option>每个选项</option>

    • 文件域

      • type=”file”
