水平居中、垂直居中以及水平垂直居中的方法整理

发表于 2019-12-12 更新于 2019-12-18 分类于 CSS

水平居中

  • 行内元素水平居中： 通过 text-align: center可以实现在块级元素内部的行内元素水平居中。

  • 块级元素水平居中：

    • 将块级元素左右外边距margin-left和margin-right设置为auto
    • 使用absolute+transform，先将父元素设置为相对定位，再将子元素设置为绝对定位，向右移动子元素，移动距离为父容器的一半，最后通过向左移动子元素的一半宽度以达到水平居中
    • 使用flex+justify-content，通过CSS3中的弹性布局flex中的justify-content属性来达到水平居中，父子元素都可以
    • 使用flex+margin，通过flex将父容器设置为为flex布局，再设置子元素居中

  • 多块级元素水平居中：

    • 使用flex布局，实现水平居中
    • 使用inline-block，将水平排列的块状元素设为display:inline-block，然后在父级元素上设置text-align:center

  • 绝对定位元素水平居中：通过子元素绝对定位，外加margin: 0 auto来实现。

垂直居中

  • 行内元素垂直居中

    • 利用行高line-height
    • 利用flex布局：利用flex布局实现垂直居中，其中flex-direction: column定义侧轴方向为纵向
    • 利用表布局：利用表布局的vertical-align: middle可以实现子元素的垂直居中。

  • 块级元素垂直居中

    • 使用absolute+-margin：通过绝对定位元素距离顶部50%，并设置margin-top向上偏移元素高度的一半，就可以实现了
    • 使用absolute+transform：通过CSS3中的transform属性向Y轴反向偏移50%的方法实现垂直居中
    • 使用flex+align-items：通过设置flex布局中的属性align-items，使子元素垂直居中
    • 使用table-cell+vertical-align：通过将父元素转化为一个表格单元格显示，再通过设置vertical-align属性，使表格单元格内容垂直居中。

      水平垂直居中

  • 绝对定位与负边距实现：这种方式需要知道被垂直居中元素的高和宽，才能计算出margin值。

  • 绝对定位与margin:auto：这种方式无需知道被垂直居中元素的高和宽。

  • 绝对定位+CSS3：利用Css3的transform，可以轻松的在未知元素的高宽的情况下实现元素的垂直居中。

  • flex布局：利用flex布局，其中justify-content用于设置或检索弹性盒子元素在主轴方向上的对齐方式；而align-items属性定义flex子项在flex容器的当前行的侧轴方向上的对齐方式。

  • flex/grid+margin:auto：容器元素设为flex布局或是grid布局，子元素只要写margin: auto即可。

HTML分类及特性整理

发表于 2019-12-05 更新于 2019-12-18 分类于 HTML

块级元素

特性

  1. 块状元素一般都作为其他元素的容器，可以容纳其他内联元素和块状元素。
  2. 块状元素可以定义自己的长度和宽度及边距。
  3. 块状元素在网页中以矩形区域显示。
  4. 默认情况下，块状元素会占据一行，块状元素会按顺序自上向下排列。

    行内元素（内联元素）

    特性

  5. 宽度及边距由内容决定，只有在内容超过html的宽度时，才会换行。
  6. 和其他元素都在一行上。
  7. 设置高度height无效，可以通过line-height来设置。
  8. 只能容纳文本或其他内联元素。

    置换元素

    行内元素中的一种特殊存在，像其中的<img><input><select>等这些元素都是置换元素，浏览器会根据这些元素的属性和标签，来决定元素的具体显示内容，这些元素往往没有内容，就是一个空元素

如何使用hexo搭建自己的博客

发表于 2019-12-04 更新于 2019-12-18 分类于 其他

准备工作

以Windows 64位为例
1.安装git
首先访问git,其次点击64-bit Git for Windows Setup进行下载，最后下载完成直接进行安装即可。
2.安装node.js
先访问node,
而后下载左边的8.10.0 LTS版本，最后下载完成之后安装即可。
3.注册github账号
打开https://github.com,根据要求进行注册即可。

创建github仓库

新建一个你的用户名.github.io的仓库，创建完成之后。
配置SSH key,首先执行
cd ~/. ssh
检查本机已存在的ssh密钥。
然后执行
ssh-keygen -t rsa -C “邮件地址”
然后连续3次回车，会生成一个文件在用户目录下。
打开用户目录，找到.ssh\id_rsa.pub文件，记事本打开并复制里面的内容。
打开你的github主页，进入个人设置 -> SSH and GPG keys -> New SSH key：将刚复制的内容粘贴到key，保存。
执行
ssh -T git@github.com
检查是否成功，如果提示
Are you sure you want to continue connecting (yes/no)?
输入yes，然后看到：
Hi liuxianan! You've successfully authenticated, but GitHub does not provide shell access.
看到这个信息说明SSH已配置成功！
最后配置 git config --global user.name "liuxianan"
git config --global user.email "xxx@qq.com" 即可。

使用hexo

执行 npm install -g hexo 进行安装。
然后新建一个文件夹，cd 文件夹位置进入文件夹，执行
hexo into
hexo会自动下载一些文件到这个目录，然后继续
hexo g
hexo s
执行以上命令之后，hexo就会在public文件夹生成相关html文件。
hexo s是开启本地预览服务，打开浏览器访问 http://localhost:4000 即可看到内容。
如果你一切都配置好了
hexo d
一定要把ssh key配置好，
在配置_config.yml中有关deploy的部分，我用的Vs Code，写代码的一个软件，直接打开，找到deploy，修改。
deploy:
type: git
repository: 你的仓库地址
branch: master
然后执行
npm install hexo-deployer-git --save
这个的作用是安装一个插件,最后
hexo d
提交所有改动。

备注

以上就是用hexo搭建blog的全部步骤，至于写自己blog文章的软件，可以自己下，也可以直接hexo new “postName”进行编写，因为博主也是一个小白，就能帮你们这么多了，有不对的地方，也希望大家理解。
